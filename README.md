
# Portafolio personal

Una aplicación móvil desarrollada en Flutter con Dart. Esta aplicación es un portafolio interactivo con una interfaz moderna, organizada y altamente modular.

## Pantallas de la app 📲
La aplicación está estructurada en vistas conectadas: 
- **About me**: presentación con datos básicos, fecha de nacimiento, ubicación y trayectoria educativa.
- **Redes sociales**: enlaces de contacto profesional y personal.
- **Hobbies**: sección interactiva que detalla actividades desempeñadas por el dev (natación, videojuegos, panadería y pastelería, y aprendizaje de idiomas).
- **Proyectos**: espacio dedicado a mostrar el catálogo de proyectos desarrollados por el dev.
- **Comentarios**: apartado para la recepción de opiniones de los usuarios.
- **Firma del desarrollador**: pantalla accesible desde el AppBar de la pantalla de About Me, que contiene las herramientas técnicas y lenguajes dominados por el dev, así como su firma digital.
## Tech utilizada 💻
- Dart (como lenguaje de programación).
- Flutter SDK (como framework de desarrollo móvil).
## Dependencias utilizadas 🔗

El proyecto hace uso de las siguientes dependencias registradas en `pubspec.yaml`:
* **`curved_navigation_bar`**: proporciona una barra de navegación inferior animada con transiciones estilizadas.
* **`accordion`**: implementado para colapsar y expandir interactivamente los segmentos de la pantalla de hobbies.
* **`url_launcher`**: utilizado para la apertura de enlaces externos a redes sociales.
## Tipografías utilizadas 🔠

El proyecto usa tipografías personalizadas para su diseño visual:
* **Coolvetica.**
* **Newake.**

Todas las tipografías están disponibles en el sitio web de DaFont.
